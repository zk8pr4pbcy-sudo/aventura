# موقع أفنتورا — نسخة الإطلاق

هذه النسخة مستقلة ونظيفة وجاهزة للنشر كموقع ثابت سريع. لا تعتمد على قاعدة بيانات، وتعمل الصفحات والمحتوى الأساسي حتى عند فتحها محليًا.

## ما الذي يعمل الآن؟

- الصفحة الرئيسية وصفحات التجارب والشركات والفعاليات والخدمات والمتجر ومن نحن والمعرض والأسئلة الشائعة والتواصل والخصوصية.
- العربية والإنجليزية والإسبانية مع حفظ اختيار اللغة.
- تصميم متجاوب للجوال والكمبيوتر.
- نموذج طلب يرتب البيانات ويفتحها في واتساب للمراجعة قبل الإرسال.
- روابط البريد والهاتف وواتساب.
- ملفات محركات البحث: `robots.txt` و`sitemap.xml` والبيانات المنظمة.
- صفحة خطأ `404.html` ونشر مباشر عبر GitHub Pages.

## المعاينة محليًا

يمكن فتح `index.html` مباشرة. ولأفضل معاينة من جهاز كمبيوتر شغّل خادمًا محليًا داخل المجلد:

```bash
python3 -m http.server 8080
```

ثم افتح `http://localhost:8080`.

## النشر على GitHub Pages

1. ارفع محتويات هذا المجلد إلى جذر مستودع GitHub.
2. من إعدادات المستودع افتح **Pages**.
3. اختر النشر من الفرع الرئيسي ومن مجلد الجذر `/`.
4. أضف ملف `CNAME` لاحقًا عند بدء ربط `aventuraksa.com`.
5. أضف سجلات DNS التي يعرضها GitHub لدى مزود الدومين، ثم فعّل HTTPS بعد اكتمال الربط.

## المرحلة الثانية المقترحة

الحجز الفوري والدفع وحساب العميل ولوحة الإدارة ليست جزءًا من هذه النسخة؛ لأنها تتطلب نظامًا خلفيًا وقاعدة بيانات وسياسات دفع وإلغاء. يمكن إضافتها بعد تشغيل الموقع وجمع الطلبات الفعلية.
